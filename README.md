# GrammarGrinder

GrammarGrinder is a lightweight Android app for practicing English grammar with quick exercises and a built-in grammar guide.

## What It Does
- Practice grammar topics with focused questions
- Use mix mode for combined practice
- Open grammar guides for fast reference
- Adjust app options like theme/language/vibration

## Tech
- Android app in Java
- Gradle build system
- JSON assets for sections/questions/guides

## Run
1. Open the project in Android Studio
2. Sync Gradle
3. Run the `app` module on an emulator or device
